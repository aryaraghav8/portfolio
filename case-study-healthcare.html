Healthcare UX Case Study

Designing a scalable diagnostic workflow system for lab technicians.

Redesigning disconnected healthcare interfaces into a unified workflow-focused platform for diagnostic operations, reporting, and technician usability.

Role
Product Designer
Timeline
4 Months
Platform
Web + Mobile
Overview

The challenge

Lab technicians were switching between multiple disconnected interfaces during diagnostic workflows, increasing cognitive load and slowing report generation.

Existing systems were designed around backend structures rather than real operational workflows, making everyday tasks inefficient and error-prone.

“The biggest issue wasn't missing features — it was workflow interruption.”

Research

Understanding technician workflows

Conducted workflow observations, usability reviews, and stakeholder discussions to understand how technicians interacted with diagnostic systems during high-frequency tasks.

Workflow interruption

Users constantly switched between tools to complete a single task.

High cognitive load

Complex interfaces increased training time and operational confusion.

Mobile usability issues

Technicians struggled with touch targets and navigation during fast-paced workflows.

Early workflow mapping and task analysis sessions.
Process

Restructuring the diagnostic workflow

The redesign focused on reducing friction across the complete operational journey:

Step 01 — Workflow simplification

Reduced unnecessary task switching and grouped related actions contextually.

Step 02 — Mobile-first operational UX

Optimized interfaces for single-handed technician usage.

Step 03 — Unified reporting structure

Created consistent reporting patterns across systems.

Wireframes and workflow restructuring explorations.
Design System

Building reusable interface patterns

Created reusable components aligned with frontend implementation to improve scalability and consistency across products.

Reusable UI components

Diagnostic cards, reporting modules, table systems, form controls, and status indicators.

Frontend-aware implementation

Component structures aligned with React architecture and responsive layout systems.

Outcomes

Improving operational efficiency

The redesigned workflow reduced friction during diagnostic operations while improving usability consistency across web and mobile systems.

Reflection

Key learnings

This project reinforced the importance of workflow-first UX in operational healthcare environments where usability directly impacts efficiency and user confidence.

It also strengthened collaboration between design and frontend implementation by aligning reusable systems early in the process.
